# QR Code Generator

A simple and responsive QR Code Generator built using HTML, CSS and JavaScript.

## 🚀 Features
- Generate QR codes for text or URLs
- Input validation
- Responsive design
- Smooth UI animation

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- QR Server API
